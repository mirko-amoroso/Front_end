Operatori

tramite degli operatori posso cobinare valori tra di loro

gli operatori determinano il risultato di un'operazione

clssificazione degli operatori in base al numero di valori che possono combinare: unaru, binari, ternario

operazioni aritmetiche

operatori binari perche combinano due valori (numerici)


+ addizione
- sottrazione
/ divisione
* moltiplicazione
% resto

gli operatori aritmetici possono anche essere unari, cioè combinano un solo valore

++ incremento
-- decremento
- negazione -> -x = -5 se ci sono due valori - è sottrazione, se c'è un solo valore - è negazione
-> incremento e decremento possono essere applicati solo a delle variabili
-> la posizione dell'operatore -> se l'operatore è a sinistra dell'operando l'operazione viene eseguita prima di valuatare l'espressione
-> se l'operatore è a destra dell'operando l'operazione viene eseguita dopo

operatori logici

consento di combinare espressioni booleane (true | false)


&& -> operatore binario -> and
|| -> operatore binario -> or
! -> operatore unario -> not

operatore terniario

perette di combinare tre dati tra di loro . restituisce un valore in base alla valutazione di un'espressione booleano
sintassi: condizione? valore1 : valore2
verifico se la condizione è vera (true | false) -> se è true restituisce valore1 senno valore2

operatori relazionali

utilizzat per confrontare dei valori

operatore none
> Maggiore
< Minore
>= Maggiore o Uguale
<= Minore o Uguale
== Uguale
!= Diverso
