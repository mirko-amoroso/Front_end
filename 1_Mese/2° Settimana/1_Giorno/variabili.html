Le variabili

servono a memorizzare dei valori

ogni variabile corrispomnde ad un nome che deve seguire delle regole

  • niente spazi, niente trattini, niente caratteri speciali
  • deve iniziare con una lettera, sono ammessi underscore _ e dollaro $
  • sono case sensitive
  • non può iniziare con un numero

dichiarazioni implicita delle variabili

non è obbligatorio dichiarare una variabile, cioè fare una operazione esplicita di creazione
quando c'è l'identificazione = l'engine crea implicitamente la variabile se non esiste già

Var, Let, Const

VAR -> è accessibile in qualque parte dello script

LET -> è accesibile solo in un determinato ambito (scope) delimitato da un blocco di codice

Un blocco di codice js è una seriedi istruzione delimitate da {}

CONST -> inserisce l'engine che il valore inserito non è modificabile

strict mode

è una opzione di controllo del codice, se viene dichiarato lo strict mode js viene eseguito con regole + stringenti -> ad esempio se una variabile non viene dichiarata ci viene mostrato errore -> se dichiaro lo strict mode -> regole meno permissive -> stile di programma più strutturato
