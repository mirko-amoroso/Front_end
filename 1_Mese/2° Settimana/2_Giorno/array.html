Array

l'array è una variabile multidimensionale. Dentro a una variabile posso inserire un valore, dentro una array posso inserire più valori

dentro una array incontriamo una struttura di dati omogenee

dentro possiamo conservare qualsiasi tipo di dato (primitivi o oggeti)

gli array sono elenchi, liste in cui salvare diversi dati

dentro una array associamo a un unico nome più valori

l'uso delle array ci semplifica le operazioni nel caso in cui dobbiamo definire più variabili

proprietà delle array

in javascript esistono una serie di metodi che ci permettono di manipolare sugli array, la sintassi per fare questa cosa è: nome_array.nome_metodo

il prio metodo -> length -> conta quanti elementi ci sono in una array

inserire | rimuovere elemento in una array

per inserire degli elementi in una array abbiamo un metodo che si chiama push

per rimuovere elemnti da una array abbiamo il metodo .pop

Altri metodi: unshift aggiunge nuovi elemnti all'inizio dell'array; shift rimuove dall'inizio dell'array shift e unshift sovrascrivono l'array -> gli indici vengono modificati
