Logica Condizionale

qualsiasi nostro script è una sequenza di istruzione vengono eseguite dall'engine javascript
tutte le nostre istruzioni (quello che scriviamo nei nostri) sono un flusso di comandi
l'insieme dei comandi che scriviamo porta la nostra pagina web da uno stato iniziale a uno stato finale, con il flusso dei comandi possiamo far ragiungere un certo risultato
avremo bisogno di istruzioni per stabile il flusso di esecuzioni di un programma

blocco codice

Le istruzioni vengono raccolte in blocchi che fungono da singole istruzioni. i blocchi di codice ci permettono di eseguire più istruzioni interamente
tipico della logica condizionale è eseguire dei blocchi di codice in seguito alla valutazione di una condizione
un blocco di codice è racchiuso da parentisi graffe {}

la logica condizionata è una categoria di istruzioni che consentono di eseguire blocchi di codice sulla base della verifica di una condizione Lo script "decide" se eseguire il blocco di codice, la decisione è basata sulla verifica di una condizione

JS prevede due istruzioni condizinali

  • if
  • switch

il costrutto if prevede tre forme diverse

  • if semplice
  • if con alternativa (if, else)
  • if a cascata (if, else.. if, else)

if semplice

la forma if semplice esguire blocco di codice solo se una condizione è vera
se la condizione non è vera il blocco di codice non viene eseguito.

schema sintattico del costrutto if semplice

If ... else

in questa forma viene eseguito un blocco di codice oppure un'altro blocco di codice in base al valore della co dizione
