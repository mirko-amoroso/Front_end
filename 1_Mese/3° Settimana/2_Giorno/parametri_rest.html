Parametri rest

rest in inglese significa resto | restante. Con Paramatro Rest raccogliamo un numero non definito di parametri in un array

quando dichairiamo una funzione indichiamo il numero di parametri -> quelli che tendezialmente valorizziamo
può capitare in alcune funzioni di vedere n parametri i parametri delle funzioni vengono inseriti in array arguments
con i parametri rest... possiamo racchiudere i parametri di troppo
