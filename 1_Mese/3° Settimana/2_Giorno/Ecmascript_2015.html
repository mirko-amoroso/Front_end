Ecmascript

Ecmascript (o ES) è lo standard per il linguaggio di scripting secondo le specifiche del documento ECMA-265(vedi sito). Ecma è la realtà di rifrimento per l a standardizzazione del settore informatico. Ecmascript è uno standard pensato per garantire l'interoperabilità delle pagine sul browser.
con l'avvento di Ecmascript sono state introdotte una serie di novità

  • Arrow e function
  • Nuovi metodi della libreria Math
  • Vedremo oggi
  • Operatore spread
  • Paramtri rest
  • Destrutturare array | oggetti
  • Template literals
  • Nuovi metodi delle stringhe
  • Nuovi metodi delle array
